# Conversor CSV - YAML

## Resumen
Script realizado en Shell Script que convierte archivos CSV a YAML  

## Modo de Uso

Primero se debe ejecutar el generador de ejemplos de prueba:

./generador.sh  (esto generara un archivo llamado "archivo.csv") 

Luego para convertirlo a YAML se debe ejecutar 

./script.sh archivo.csv > archivo.yaml
